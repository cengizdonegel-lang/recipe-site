About This Project

This website was created as part of the AI Builder Challenge. The goal was to use an AI tool to design and build a simple personal website from scratch using HTML, CSS, and JavaScript.

I chose to create a recipe website because cooking is something I enjoy, and recipes are a great example of simple, practical content that everyone understands.

The site is structured in a component-based way, with separate folders for header, footer, and reusable UI elements. This makes the project easy to scale and maintain.

Technologies Used

  • HTML5
  • CSS3 (component-based structure)
  • JavaScript (basic interactivity)
  • AI-assisted coding
